# Class Performance Analyzer

## Project Overview

Class Performance Analyzer is a Python menu-driven application developed as a mini project while learning Python functions. The program analyzes the performance of two classes in a selected subject by collecting students' marks and calculating basic statistical measures.

The project demonstrates the practical use of functions, loops, conditional statements, lists, exception handling, and user input validation in a real-world educational scenario.

---

## Features

* Menu-driven interface
* Analyze one subject at a time
* Supports the following subjects:

  * English
  * Math
  * Science
  * Islamiyat
  * Urdu
  * Pak Study
* Accepts marks for 10 students from each class
* Validates marks (0–100)
* Handles invalid input using exception handling
* Calculates:

  * Mean
  * Median
  * Mode
* Compares the performance of both classes
* Displays a final performance report

## Project Workflow

1. Display the main menu.
2. User selects a subject.
3. Enter marks for 10 students of Class 1.
4. Enter marks for 10 students of Class 2.
5. Validate all entered marks.
6. Calculate the mean, median, and mode for each class.
7. Compare the average performance of both classes.
8. Display the complete analysis report.
9. Return to the main menu until the user selects Exit.
