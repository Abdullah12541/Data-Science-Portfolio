A menu-driven calculator developed in Python using functions, conditional statements, loops, and exception handling.

- Subtraction
- Multiplication
- Division
- Remainder
- Square
- Cube

- Menu validation

- Return Values
- If-Elif-Else
- While Loops
- Try-Except
- User Input
- Error Handling
