# Model

## Purpose
This folder contains the trained YOLOv10 model used for object detection.

## Model Details
- Model: YOLOv10
- Framework: PyTorch
- Training Environment: Google Colab
- GPU: Tesla T4

## Final Selected Model
The 20-epoch model was selected as the final model because it achieved the best overall performance among all experiments.

## Performance
- mAP50: 0.9880
- mAP50-95: 0.9710

## Contents
This folder stores:
- Trained model weights (.pt file)
- Model-related documentation
- Model performance information
