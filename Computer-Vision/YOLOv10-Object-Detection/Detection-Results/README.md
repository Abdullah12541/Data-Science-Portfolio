# Detection Results

## Purpose
This folder contains object detection results generated by the trained YOLOv10 model on unseen images.

## Detected Classes
- Apple
- Banana
- Orange
- Potato

## Contents
- Apple_Detection.jpg
- Banana_Detection.jpg
- Orange_Detection.jpg
- Potato_Detection.jpg

## Description
Each image contains:
- Predicted class label
- Confidence score
- Bounding box around the detected object

These results demonstrate the performance of the trained YOLOv10 model on unseen test images.
