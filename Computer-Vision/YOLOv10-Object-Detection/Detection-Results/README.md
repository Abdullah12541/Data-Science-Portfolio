# Detection Results

## Purpose
This folder contains object detection results generated by the trained YOLOv10 model on unseen test images.

## Detected Classes
- Apple
- Banana
- Orange
- Potato

## Contents
The folder will contain prediction images showing:
- Detected objects
- Bounding boxes
- Class labels
- Confidence scores

## Testing
The model was evaluated on unseen images to verify its object detection performance after training.
