# Training Results

## Purpose
This folder contains the outputs generated during the training and evaluation of the YOLOv10 object detection model.

## Included Results
- Training graphs
- Validation graphs
- Precision and Recall curves
- F1-Score curve
- Confusion Matrix
- mAP50 results
- mAP50-95 results
- Epoch comparison results

## Experiments
The model was trained and evaluated using multiple epoch settings:

- 10 Epochs
- 20 Epochs
- 30 Epochs

## Best Model Performance
- Best Epoch Setting: 20 Epochs
- mAP50: 0.9880
- mAP50-95: 0.9710

## Contents
This folder will store training visualizations, performance metrics, and evaluation outputs used for project analysis and comparison.
