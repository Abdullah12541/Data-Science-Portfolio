## ⭐ Featured Project

### Image-Based Object Detection Using YOLOv10 with Multi-Epoch Performance Evaluation

This project focuses on object detection using YOLOv10 and evaluates model performance across multiple training epochs. The project includes dataset preparation, model training, testing, performance evaluation, and comparative analysis.

#### Project Highlights

* Dataset Preparation and Augmentation
* YOLOv10 Model Training
* Multi-Epoch Performance Evaluation
* Precision, Recall, and mAP Analysis
* Testing on Unseen Images
* Detection Result Visualization
* Comparative Performance Study
